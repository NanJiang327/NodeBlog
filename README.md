# Node Blog (express 4.0)
Simple blog based on node.js + express + ejs + mongodb

This is a learning example app.

# Build Setup
``` bash
# install dependencies
npm install

#server with hot reload at localhost:3000
nodemon index.js
```

# Tech stack
- Node.js
- Express 4
- ejs
- MongoDB
- Mongolass
- Marked (Markdown to HTML parser)
- jQuery
- Semantic-ui
